English API Usage Guide

This section documents which command-line and Python APIs are used for HDF5 export, training, prediction, and diagnostics. It also maps each workflow to the source files that implement it, so the generated HTML can be used as a guide for reading the code.

For code study, start with Code Reading Map, then read Code walkthrough for the GNN model, message passing, and loss definitions.
